HTML

  • The head element contains information about the webpage
  • The body element represents the visible content shown to the user
  • Use h1-6 for different sizes of headings on the webpage
  • You can make unordered lists and ordered lists using ul or ol and listing items with li
  • Add images using the img element and using src attribute to define the location of the image
  • Remember to add alt attribute to provide alt text for screen readers and improve search engine rankings for your website
  • Always remember to close your elements and attributes
  • You can add a hyperlink by using the anchor, a, element and the href attribute to point to the URL for the link

CSS

  • Declarations: property we want to apply and value of that property [i.e. color(property): blue(value)]
  • CSS Box Model! Margin>Border>Padding>Content
  • Selector tells which elements to apply the rules to (* is for all elements visible on the page)

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch name: creates a new branch and switches to it
  • Use git add -A to commit all files in the working branch
  • We use the -m flag to associate a message with our commit. Commit messages should be short descriptive messages of the changes implemented

JavaScript

  • Variable: named container that allows us to store data in our code
  • var is used to declare a variable
  • Use for loops to navigate arrays
  • For Loops have 3 important statements:
  • 1. determine starting point (declare variable and assign your preferred value i.e. var x = 0;)
  • 2. set condition (x < shapes.length;)
  • 3. allow array to iterate over each time (x+1 or x++)
  • Function: is simply a set of instructions that tells the computer how to perform a certain task.
  • A function's name is followed by parentheses
  • You must call a funtion after defining it in order to execute the function
